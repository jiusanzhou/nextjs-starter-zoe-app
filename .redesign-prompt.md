### 1. Projects Section (src/components/sections/projects-section.tsx + src/components/project-card.tsx)

  - 项目名用更大的字号
  - 描述文字更突出

### 2. Posts Section (src/components/sections/posts-section.tsx + src/components/post-card.tsx)

当前问题：PostCard 使用 shadcn Card 组件，比较传统；日期格式硬编码中文。

重新设计：
- PostCard (grid 模式):
  - 更现代的设计，不依赖 shadcn Card
  - 大封面图区域（如果有 banner）

  - hover 效果：整体向上浮起 + 阴影
- PostTile (tile 模式):
  - 更简洁的列表项
  - 去掉 "No Image" 占位
  - hover 高亮效果更明显
- "查看更多" 按钮文字国际化（从 labels 读取）
- "暂无文章" 空状态国际化
- 2列或3列自适应，根据有无 banner 图自动选择

### 3. Contact Section (src/components/sections/contact-section.tsx + src/components/author-card.tsx)

当前问题：只是包裹了一个 AuthorCard，完全是个人站风格。

重新设计为通用的 Contact/CTA section：
- 支持两种模式（通过配置控制）：
  a) 表单模式（产品站）：简洁的联系表单（姓名、邮箱、消息），可以只是展示用的 UI，不需要实际提交
  b) 卡片模式（个人站）：保留当前 AuthorCard 的能力
- Contact section 配置扩展：
  ```typescript
  interface ContactSection {
    type: 'contact';
    title?: string;
    description?: string;
    style?: 'form' | 'card' | 'simple';  // 新增
    email?: string;      // 展示联系邮箱
    links?: { icon?: string; title: string; href: string; description?: string }[];  // 联系方式列表
  }
  ```
- style: 'form' — 左边联系信息+链接列表，右边表单
- style: 'card' — 当前的 AuthorCard（保留）
- style: 'simple' — 只显示标题+描述+联系方式链接

在 src/types/config.ts 中更新 ContactSection 类型。
