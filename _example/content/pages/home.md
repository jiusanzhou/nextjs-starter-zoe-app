这是一个 [Next.js](https://nextjs.org/) 的 Starter。
同时也是一个快速建立站点的工具。你可以在 [Github](https://github.com/jiusanzhou/nextjs-starter-zoe-app)
上看到所有的代码。

如果你只是想建立网站，可以直接使用快速安装脚本：

```bash
curl -sSL https://raw.githubusercontent.com/jiusanzhou/nextjs-starter-zoe-app/main/scripts/zoe-site.sh | bash -s new my-site
```

## 理念

> **任何业务逻辑都应向[「无码平台」](https://wuma.dev)演进。**

## 特性

- 🎨 **7 种内置主题** - default, cyber, minimal, apple, github, vercel, stripe
- 📝 **YAML 配置** - 单个 `zoe-site.yaml` 文件配置整个站点
- 📂 **零配置内容** - 只需将 Markdown 文件放入 `content/` 目录
- 🚀 **现代技术栈** - Next.js 16, React 19, TypeScript, Tailwind CSS 4
- 💎 **美观 UI** - shadcn/ui 组件 (Radix + Tailwind)
- 🔍 **SEO 友好** - 自动生成元数据
- 🌙 **暗黑模式** - 内置明暗主题切换
- 📰 **博客系统** - 文章、标签、归档开箱即用
- 🗂️ **项目展示** - 美观地展示你的项目
- 📡 **RSS 订阅** - 自动生成 RSS 支持
- 📦 **Git 内容** - 从远程 Git 仓库加载内容
- 📊 **数据分析** - Google Analytics 和 Plausible 支持
- 📥 **应用发布** - 从 GitHub/Gitee 获取并展示 Release
