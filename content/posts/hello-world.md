## 特性

这个模板包含以下特性：

- 🚀 **Next.js 16** - 最新的 React 框架
- 🎨 **shadcn/ui** - 美观的 UI 组件
- 📝 **MDX** - 支持在 Markdown 中使用 React 组件
- 🌙 **暗色模式** - 自动适配系统主题
- 📱 **响应式** - 完美适配各种设备
